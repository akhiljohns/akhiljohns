## 🚀 My Approach

- **Feature-driven architecture** for scalable codebases  
- **Strict TypeScript** and schema-based validation  
- Clear separation of **server state, client state, and form state**  
- Focus on **performance, maintainability, and readability**  

I use **AI tools across my workflow** — from planning and architecture discussions to implementation, refactoring, and edge-case validation. This helps me ship high-quality solutions faster without compromising structure or code standards.
